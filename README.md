# zjunet

Command Line Scripts for ZJU (VPN / WLAN / DNS)

## Features

- ZJU VPN (l2tp)

- ZJU WLAN

- 路由器支持（written in Bourne Shell）

- 多拨支持（多账户负载均衡）

- ZJUWLAN 与 ZJUVPN 带宽叠加（1 WLAN + N VPN，在 ZJUWLAN 下）

- DNS 自动测试设置（适用于 DNS 坏掉的情况）

## Requirements

### vpn

- xl2tpd

### wlan

- curl

- iconv
